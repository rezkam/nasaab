# Nasaab

*Persian for "installer"*

Short scripts to setup my working system on macOS.

## Scripts

### setup-gnu-tools.sh

Installs GNU tools and utilities on macOS via Homebrew.

**What it installs:**

GNU tools:
- coreutils (ls, cp, mv, rm, cat, date, etc. - 100+ utilities)
- sed, grep, find, awk, tar, make, diff, bash 5.x

Other utilities:
- wget, watch, git, less, tmux, ripgrep
